## 📌 About The Project  

FlipInIndia is a **dynamic business website** designed to showcase services, manage client interactions, and handle pickup requests efficiently.

It combines:
- 🌐 Responsive frontend  
- 🖼️ Interactive gallery  
- 🛠️ Powerful admin dashboard  

---

## ✨ Key Features  

### 🌐 Frontend  
- Multi-page website:
  - Home  
  - About Us  
  - Services  
  - Gallery  
  - Contact  
- Fully responsive design  
- Dynamic gallery with factory images  
- Contact form (stores data in DB)  
- Pickup request system  

---

### ⚙️ Admin Panel  
- Secure login & session management  
- Manage:
  - 📩 Contacts  
  - 🚚 Pickup requests  
  - 🧾 Testimonials  
  - 🛍️ Store & banners  
- Profile management  

---

### 🧠 Backend Highlights  
- Modular PHP architecture  
- MySQL database integration  
- Clean routing using `.htaccess`  
- Config files:
  - `config.php`  
  - `database.php`  
- Prebuilt database dump included
